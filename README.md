# IKEBUKURO 3D — 池袋ファーストパーソン

池袋を舞台にした**3D・一人称視点（FPS）**のGTA風オープンワールドゲーム。**iPhoneのSafariで開くだけ**で動きます（WebGL / Three.js）。

## 🔗 オンラインで遊ぶ
**https://poinkare-kota.github.io/poinkare-claude/**

## ローカルで遊ぶ
`index.html` をブラウザで開くだけ。ローカルサーバ例:
```bash
python3 -m http.server 8000
# → http://localhost:8000
```
※ 3Dエンジン（Three.js）をCDN（jsDelivr）からESモジュールで読み込むため、初回はネット接続が必要です。

## 操作
**タッチ（iPhone / 横向き推奨）**
| 操作 | 内容 |
|------|------|
| 左スティック | 移動（徒歩）／前後・ステア（運転） |
| 画面右をドラッグ | 視点を回す（見回し） |
| 🔫 ボタン | 撃つ |
| **A** ボタン | 近くの車に乗る・降りる |
| **B** ボタン | 走る／車のブースト |
| **⏸** ボタン | 一時停止 |

**デスクトップ**：WASD移動、マウスで視点（クリックでポインタロック＆射撃）、E=乗降、Shift=走る、Esc=ポーズ。

## 特徴
- 🏙 **一人称で歩ける実在の池袋** — 池袋駅・JR線路・西口公園(IWGP/噴水)・東京芸術劇場・サンシャイン60（超高層）・サンシャインシティ・乙女ロード・東武/西武百貨店・PARCO・ビックカメラ・明治通り・グリーン大通り
- ☀️ **太陽光と影・昼夜サイクル** — 時間が流れ、夕暮れ〜夜は空と霧が変化。窓明かり・街灯・歓楽街のネオンが点灯
- 🔫 **射撃**（レイキャスト＋マズルフラッシュ＋トレーサー）/ 🚗 **車の強奪と運転**（コックピット視点）
- 🚓 **手配度システム（★0〜5）** — 犯罪で警官が出動し追跡・発砲。時間で沈静化
- 👥 **3Dの群衆と交通** — 歩行アニメ付きの歩行者、危険を察知して逃げる挙動、行き交う車
- 🗺 **円形ミニマップ / ミッション**（目的地ビーム・報酬）/ HUD（時刻・所持金・体力・現在地）

## 技術
- `index.html` + `style.css` + `game3d.js`（Three.js r160 を CDN から `importmap` で読み込み）
- 疑似PBRマテリアル、ディレクショナル太陽光＋ソフトシャドウ、霧による奥行き、proceduralな窓/ネオンの発光マップ
- iPhoneのノッチ／セーフエリア対応、`touch-action` 無効化、デバイスピクセル比キャップ（性能配慮）

> 旧バージョン（2Dトップダウン）の実装は `game.js` に残しています。
