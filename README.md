# IKEBUKURO STORIES — 池袋ストリート

池袋を舞台にしたGTA風オープンワールドゲーム。**iPhoneのSafariで開くだけ**で動く、依存ライブラリなしの自己完結型 HTML5 Canvas ゲームです。

## 遊び方

`index.html` をブラウザで開くだけ（オフライン動作可）。ローカルで試す場合:

```bash
python3 -m http.server 8000
# → ブラウザで http://localhost:8000 を開く
```

iPhoneでは **横向き** を推奨。「▶ ゲーム開始」をタップしてスタート。

## 操作（タッチ）

| 操作 | 内容 |
|------|------|
| 左スティック | 移動（徒歩）／前後・ステアリング（運転） |
| **A** ボタン | 近くの車に乗る・降りる／徒歩時はパンチ |
| **B** ボタン | ダッシュ／車のブースト |
| **⏸** ボタン | 一時停止 |

PCではキーボード（WASD / 矢印キー、E=乗降・殴る、Shift=ダッシュ、Space=A）でも操作可能。

## 特徴

- 🏙 **実在の池袋を再現** — 池袋駅・JR線路・西口公園(IWGP)・東京芸術劇場・サンシャイン60／サンシャインシティ・乙女ロード・東武／西武百貨店・PARCO・ビックカメラ・明治通り・グリーン大通りなど
- 🧍 **キャラクター造形** — 頭・髪・胴・腕・脚を個別に描画し、歩行アニメ・影・肌や服のバリエーションで群衆を表現
- 🚗 **車の強奪と運転** — 慣性のあるドライビング、車種（一般車／タクシー／トラック／パトカー）
- 🚓 **手配度システム（★0〜5）** — 犯罪で星が増え、パトカーが出動・追跡・発砲。時間で沈静化
- 🌗 **昼夜サイクル** — 時間が流れ、夜は窓明かり・街灯・ヘッドライト・歓楽街のネオンが点灯
- 👥 **生きた街** — 行き交う歩行者と交通、危険を察知して逃げる群衆
- 🗺 **ミニマップ & ミッション** — 円形ミニマップ、目的地マーカー、報酬付きの連続ミッション

## 技術

- 単一の `index.html` + `style.css` + `game.js`（外部CDN・フレームワーク不使用）
- 疑似3Dトップダウン描画（建物の高さ・側面・屋上・窓を立体表現）
- iPhoneのノッチ／セーフエリア対応、`touch-action` 無効化でスクロール抑止、デバイスピクセル比対応
