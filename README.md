## 📘 内部管理責任者 学習アプリ「内管マスター」

> **スマホ版**：**[study.html](study.html)**（iPhone向け・演出多めのゲーム風）
> **PC版**：**[study-pc.html](study-pc.html)**（職場向け・落ち着いた資料ツール風／キーボード操作対応）
> どちらも学習データ・進捗を共有。CDN不要・完全オフライン動作。

日本証券業協会「内部管理責任者資格試験」向けの学習＆暗記アプリです。
公式テキスト「営業責任者・内部管理責任者必携」の章立てと、実際の模擬問題をもとに構成しています。

- **学習パート**：公式テキスト準拠の**9単元（全55セクション）**を、**色分けハイライト**（🔥頻出・<key>重要語</key>・数字バッジ）と**SVGポンチ絵**で図解し、画像記憶をサポート
  1. 内部管理態勢の全体像 / 2. 役職員の職業倫理 / 3. 顧客口座の開設 /
  4. 投資勧誘の管理 / 5. 顧客注文の受託の管理 / 6. 受渡し・保管等の管理 /
  7. 協会員と役職員の規制 / 8. 不公正取引の規制等 / 9. その他内部管理に関する事項
- **テストパート**：実試験と同形式の**○×・5択**を**全149問**収録。**単元ごと**または**全範囲ランダム**で出題し、各問に解説と必携の参照ページ付き
- **ドーパミン演出**：Web Audio APIで生成する効果音、**コンボ・XP・レベルアップ・紙吹雪・画面フラッシュ**などチープなスマホゲー風の演出
- 間違えた問題は**「まちがい復習」**に自動でストック。進捗・XPは端末に保存（localStorage）

> ⚠️ 内容は学習用にまとめたもので、最新・正確な規則は日本証券業協会の公式テキスト等で必ずご確認ください。

# IKEBUKURO 3D — 池袋ファーストパーソン

池袋を舞台にした**3D・一人称視点（FPS）**のGTA風オープンワールドゲーム。**iPhoneのSafariで開くだけ**で動きます（WebGL / Three.js）。

## 🔗 オンラインで遊ぶ
**https://poinkare-kota.github.io/poinkare-claude/**

## ローカルで遊ぶ
`index.html` をブラウザで開くだけ。ローカルサーバ例:
```bash
python3 -m http.server 8000
# → http://localhost:8000
```
※ 3Dエンジン（Three.js）をCDN（jsDelivr）からESモジュールで読み込むため、初回はネット接続が必要です。

## 操作
**タッチ（iPhone / 横向き推奨）**
| 操作 | 内容 |
|------|------|
| 左スティック | 移動（徒歩）／前後・ステア（運転） |
| 画面右をドラッグ | 視点を回す（見回し） |
| 🔫 ボタン | 撃つ |
| **A** ボタン | 近くの車に乗る・降りる |
| **B** ボタン | 走る／車のブースト |
| **⏸** ボタン | 一時停止 |

**デスクトップ**：WASD移動、マウスで視点（クリックでポインタロック＆射撃）、E=乗降、Shift=走る、Esc=ポーズ。

## 特徴
- 🏙 **一人称で歩ける実在の池袋** — 池袋駅・JR線路・西口公園(IWGP/噴水)・東京芸術劇場・サンシャイン60（超高層）・サンシャインシティ・乙女ロード・東武/西武百貨店・PARCO・ビックカメラ・明治通り・グリーン大通り
- ☀️ **太陽光と影・昼夜サイクル** — 時間が流れ、夕暮れ〜夜は空と霧が変化。窓明かり・街灯・歓楽街のネオンが点灯
- 🔫 **射撃**（レイキャスト＋マズルフラッシュ＋トレーサー）/ 🚗 **車の強奪と運転**（コックピット視点）
- 🚓 **手配度システム（★0〜5）** — 犯罪で警官が出動し追跡・発砲。時間で沈静化
- 👥 **3Dの群衆と交通** — 歩行アニメ付きの歩行者、危険を察知して逃げる挙動、行き交う車
- 🗺 **円形ミニマップ / ミッション**（目的地ビーム・報酬）/ HUD（時刻・所持金・体力・現在地）

## 高画質化（実3Dモデル＋ライティング）
- **実物のリグ付き人物モデル**（歩行アニメ付き）と**車モデル**をCDNから読み込み（読込失敗時は手書きモデルへ自動フォールバックして必ず起動）
- **IBL環境反射**（RoomEnvironment）でPBRマテリアルがリアルに反射、**本格的な空**（Sky）＋太陽位置を時間に同期
- **ブルーム＋ACESトーンマッピング**のポストプロセス（夜のネオンが発光）、**ソフトシャドウ**、クリアコート車体塗装
- ※ ブラウザ／スマホ実行のため、実物GTA Vのような写実フォトリアルとの完全一致はできません（その手前の「リアル寄り」を目指した実装です）

## 技術
- `index.html` + `style.css` + `game3d.js`（Three.js r160 ＋ addons を CDN から `importmap` で読み込み）
- iPhoneのノッチ／セーフエリア対応、`touch-action` 無効化、デバイスピクセル比キャップ（性能配慮）

> 旧バージョン（2Dトップダウン）の実装は `game.js` に残しています。
